# Exercício 3

Faça um programa que calcule a seguinte equação: y = 3x² - 5x + 13

Armazene x no registrador $t5 com a instrução addi $t5, $zero, x, substituindo x
pelo valor desejado, e sempre que precisar o valor de x, utilize o valor
armazenado no registrador $t5. Armazene o resultado y no registrador $t6. Faça
teste com diferentes valores positivos e negativos de x.
